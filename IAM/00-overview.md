# Overview

This lab captures an enterprise-style AWS federation setup.
The focus is on clarity, repeatability, and security-minded configuration.

## Scope (so far)
- AWS account baseline IAM configuration
- Identity Provider integration approach (IdP choice + rationale)
- Federation flow validation
- Troubleshooting notes encountered during setup

## Why this exists
- Build a reusable reference for future architecture work
- Create a timestamped, credible public artifact of hands-on skills
